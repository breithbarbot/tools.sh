# Tools.sh
> Tools for symfony 3

<br>

## Step 1 - File executable on UNIX (755) :

    chmod +x tools.sh

<br>

## Step 2 (Optional) - Convert file `dos` to `unix` :
If you have a message "Aucun fichier ou dossier de ce type", run :

    apt-get install dos2unix

And run : `dos2unix tools.sh`

<br>

## Step 3 - Run :

    ./tools.sh
